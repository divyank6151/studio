# **App Name**: MarketPulse Pro

## Core Features:

- Real-Time Market Tracker: Display live or near-real-time data for DXY, Gold, and Crude Oil, including line charts, percentage change, daily high/low, and color-coded movement indicators.
- Automated Daily Global Briefing: Generate a daily AI-powered summary of news from Bloomberg, Reuters, Financial Times, CNBC, and Wall Street Journal, focusing on macro economy, commodities, and currency movements.
- YouTube Sentiment Intelligence: Analyze YouTube comments from videos to identify hidden content ideas, audience pain points, and emotion heatmaps.
- Newsletter & Blog Engine: Generate SEO-optimized blog posts and newsletter versions from video scripts or transcripts using AI.
- Private Community Portal: Create a paid, members-only portal with exclusive research notes, downloadable PDFs, and a discussion section.
- Workflow Automation: Integrate with Notion and Slack for research sync and alerts. Trigger email briefings and notifications based on market events and new video uploads.
- Sentiment analysis tool: The tool should decide whether sentiment is relevant when processing the comments in youtube. The model should decide if there's relevant pain points or comments to summarize for content.

## Style Guidelines:

- Primary color: Deep blue (#2E5266) to evoke trust and professionalism.
- Background color: Light gray (#F0F4F7) for a clean, modern look.
- Accent color: Soft teal (#6BBBA7) for highlighting key data points and CTAs.
- Body font: 'PT Sans', a humanist sans-serif with a blend of modern look and slight warmth.
- Headline font: 'Space Grotesk' a sans-serif with a computerized feel; pairs well with 'PT Sans'.
- Use clear, professional icons to represent market trends and data points.
- Implement a clean, single-page layout with expandable sections for detailed charts and briefings.
- Incorporate subtle animations for data loading and updates to enhance user experience.
